## Anti-Patterns Verdict

Does not look AI-generated. The interface has genuine authorship. The warm stone palette with amber accent is considered and distinctive. The two-mode contrast is a real design decision. The StriaLogo SVG is bespoke. The keyboard-first density shows product thinking.

Concerns: Home page tool grid is 6 identical cards with icon + title + description (the project's own banned pattern). Some inline styles feel rushed (RightRail hardcoded hex values).

## Priority Issues

### [P1] Destructive actions lack confirmation
Apply Redactions permanently burns content with no undo and no confirmation dialog. Clear all annotations is immediate and irreversible. Apply Crop is destructive with no warning.

### [P1] Stale port 7341 in backend-offline tooltip
Line 923 of Viewer.tsx still says port 7341. User-facing recovery message points to wrong port.

### [P2] Home page uses the project's own banned card-grid pattern
6 identical tool cards in a 3-column grid. DESIGN.md explicitly bans this pattern.

### [P2] Green download button and blue crop border break the color system
Download uses bg-green-600 (not in design system). Crop border uses #3b82f6 (blue, violates Warm-Not-Blue rule).

### [P2] Accessibility is minimal (3 ARIA attributes across entire codebase)
No aria-label on buttons. No aria-live regions. No role attributes on custom interactions. No skip-nav. Color-only status signaling. Does not meet WCAG 2.1 AA commitment.
